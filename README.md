# Calcifer

## About Calcifer

Calcifer is a private [discord](discordapp.com) bot written in C# using [Discord.Net](https://github.com/discord-net/Discord.Net) on [.Net Core version 3.0.0-preview.19073.11](https://github.com/dotnet/core)

This bot is not really intended for use outside of the couple of servers I made it for. The code is probably sub par and should only be used as a very basic example.
