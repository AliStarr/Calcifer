# Calcifer

## About Calcifer

Calcifer is a private [discord](discordapp.com) bot written in C# using [Discord.Net](https://github.com/discord-net/Discord.Net) on [.Net 5.0](https://dotnet.microsoft.com/download/dotnet/5.0)

This bot is not really intended for use outside of the couple of servers I made it for. The code is probably sub par and should only be used as a very basic example.
